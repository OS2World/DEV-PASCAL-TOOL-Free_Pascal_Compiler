li : list element

The tag li denotes an element in a ol or ul list. The usage is the same as for it’s HTML counterpart: It can occur only inside one of the ol or ul list tags. It’s contents may be arbitrary text and formatting tags, contrary to HTML tags, the li tag always must have a closing tag. Note that it cannot be used in a definition list (dl (108)).

Example: 

 <ul>
 <li>First item in the list</li>
 <li>Second item in the list</li>
 </ul>
Will be typeset as
  • First item in the list.
  • Second item in the list.

See also: ol (128), ul (153).
