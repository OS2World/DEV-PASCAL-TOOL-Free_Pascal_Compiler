input

This option is identical in meaning and functionality as the input option for FPDOC. (section 3.2, page 41) It specifies the Pascal unit source file that will be scanned and for which a skeleton description file will be generated. Multiple input options can be given, and element tags will be written for all the files, in one big output file.
