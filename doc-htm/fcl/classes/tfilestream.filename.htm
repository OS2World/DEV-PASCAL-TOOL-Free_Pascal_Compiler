TFileStream.FileName

The filename of the stream.

Declaration

public property TFileStream.FileName: String
  read FFilename;

Description

FileName is the name of the file that the stream reads from or writes to. It is the name as passed in the constructor of the stream; it cannot be changed. To write to another file, the stream must be freed and created again with the new filename.
