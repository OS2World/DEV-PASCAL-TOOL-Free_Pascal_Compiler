TStringList.Destroy

Destroys the stringlist.

Declaration

public destructor TStringList.Destroy; override;

Description

Destroy clears the stringlist, release all memory allocated for the storage of the strings, and then calls the inherited destroy method.

Remark: Any objects associated to strings in the list will not be destroyed; it is the responsability of the caller to destroy all objects associated with strings in the list.
