LineStart

Finds the start of a line in Buffer before BufPos.

Declaration

function LineStart(

  Buffer: PChar;

  BufPos: PChar

):PChar;

Description

LineStart reversely scans Buffer starting at BufPos for a linefeed character. It returns a pointer at the linefeed character.
