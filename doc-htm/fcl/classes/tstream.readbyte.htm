TStream.ReadByte

Read a byte from the stream and return its value.

Declaration

public function TStream.ReadByte: Byte;

Description

ReadByte reads one byte from the stream and returns its value.

Errors

If the byte cannot be read, a EStreamError exception will be raised. This is a utility function which symply calls the Read function.
