TAbstractObjectReader.ReadInt8

Read an 8-bit integer from the stream.

Declaration

public function TAbstractObjectReader.ReadInt8: ShortInt; virtual; abstract;

Description

ReadInt8 is called by the streaming process if it expects to read an integer value with a size of 8 bits (1 byte) from the stream (i.e. after ReadValue returned a valuetype of vaInt8). The return value is the value if the integer. Note that the size of the value in the stream does not actually have to be 1 byte.
