Inc

Declaration:
Procedure Inc (Var X : Any ordinal type[; Increment : Any ordinal type]);
Description:
Inc increases the value of X with Increment. If Increment isn’t specified, then 1 is taken as a default.
Errors:
If range checking is on, then A range check can occur, or an overflow error, when an attempt is made to increase X over its maximum value.
See also:
Dec (480)

Listing: refex/ex32.pp

Program Example32;

{ Program to demonstrate the Inc function. }

Const
  C : Cardinal  = 1;
  L : Longint   = 1;
  I : Integer   = 1;
  W : Word      = 1;
  B : Byte      = 1;
  SI : ShortInt = 1;
  CH : Char     = 'A';

begin
  Inc (C);     { C:=2    }
  Inc (L,5);   { L:=6    }
  Inc (I,-3);  { I:=-2   }
  Inc (W,3);   { W:=4    }
  Inc (B,100); { B:=101  }
  Inc (SI,-3); { Si:=-2  }
  Inc (CH,1);  { ch:='B' }
end.
