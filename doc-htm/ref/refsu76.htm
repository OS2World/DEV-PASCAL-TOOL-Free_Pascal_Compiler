Blockwrite

Declaration:
Procedure Blockwrite (Var F : File; Var Buffer; Var Count : Longint);
Description:
BlockWrite writes count records from buffer to the file F.A record is a block of bytes with size specified by the Rewrite (559) or Reset (558) statement.

If the records couldn’t be written to disk, a run-time error is generated. This behavior can be controlled by the {$i} switch.

Errors:
Depending on the state of the {$I} switch, a runtime error can be generated if there is an error. In the {$I-} state, use IOResult to check for errors.
See also:
Blockread (458),Close (465), Rewrite (559), Assign (455)

For the example, see Blockread (458).
