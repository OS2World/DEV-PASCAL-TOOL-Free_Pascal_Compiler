Dec

Declaration:
Procedure Dec (Var X : Any ordinal type[; Decrement : Any ordinal type]);
Description:
Dec decreases the value of X with Decrement. If Decrement isn’t specified, then 1 is taken as a default.
Errors:
A range check can occur, or an underflow error, if an attempt it made to decrease X below its minimum value.
See also:
Inc (513)

Listing: refex/ex14.pp

Program Example14;

{ Program to demonstrate the Dec function. }

Var
  I  : Integer;
  L  : Longint;
  W  : Word;
  B  : Byte;
  Si : ShortInt;

begin
 I:=1;
 L:=2;
 W:=3;
 B:=4;
 Si:=5;
 Dec (i);    { i:=0  }
 Dec (L,2);  { L:=0  }
 Dec (W,2);  { W:=1  }
 Dec (B,-2); { B:=6  }
 Dec (Si,0); { Si:=5 }
end.
