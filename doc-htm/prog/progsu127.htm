Stack frame omission

Under specific conditions, the stack frame (entry and exit code for the routine, see section section 6.3, page 240) will be omitted, and the variable will directly be accessed via the stack pointer.

Conditions for omission of the stack frame:

  • The function has no parameters nor local variables.
  • Routine is declared with the assembler modifier.
  • Routine is not a class.
