Rules

In this section dependency rules for the units and any other needed targets can be inserted. It will be included at the end of the generated makefile. Targets or ’default rules’ that are defined by fpcmake can be inserted here; if they are not present, then fpcmake will generate a rule that will call the generic fpc_ version. For a list of standard targets that will be defined by fpcmake, see section E.2, page 518.

For example, it is possible to define a target all:. If it is not defined, then fpcmake will generate one which simply calls fpc_all: 

 all: fpc_all
The fpc_all rule will make all targets as defined in the Target section.
